# Качество контента: дополнения для пресс-релизов

Дополняет `content-quality.md`.

## Голос пресс-релиза

- Деловой, но читаемый. Фактологический. Конкретный
- Короткие абзацы (3–5 предложений), одна мысль — один абзац
- Факты в начале, детали в конце
- Конкретные цифры вместо «значительно», «существенно»
- Активный залог: «VK Cloud запустила», не «был запущен сервис»

## Дополнительные запрещённые слова

Помимо общего списка: не имеющий аналогов, лучший на рынке, производит реализацию, спешим сообщить, с гордостью представляем, вашему вниманию, в настоящее время, в разы, в целом.

## Дополнительные замены

| Клише | Замена |
|-------|--------|
| инновационное решение | решение для [конкретная задача] |
| в разы эффективнее | эффективнее на X% |
| анонсировала запуск | запустила |

## Факты (PR)

- Минимум 2 конкретные цифры на релиз; партнёрский релиз/меморандум — по фактуре
  (может быть без цифр — суть в самом факте партнёрства, `press-release-standards.md` §2 и «Лид-абзац»)
- Цифра в цитате спикера — если релиз цифровой (продуктовый/цифровой инфоповод);
  открытие цитаты — сильнейший релевантный тезис, рыночная цифра — один из вариантов,
  не обязательный (канон требований к цитате — `press-release-standards.md` §2)
- Если данных нет — запроси у пользователя
