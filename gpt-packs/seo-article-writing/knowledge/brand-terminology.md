# Терминология: дополнения для SEO-статей

Дополняет «brand-terminology» (внутренний материал системы, в эту выгрузку не входит).

## Продуктовые названия — SEO-контекст

| Правильно | Неправильно |
|-----------|-------------|
| Cloud Servers | Виртуалки, облачные серверы (в контексте продукта) |
| Object Storage | Cloud Storage, Объектное хранилище (как имя продукта) |
| Cloud Containers | Облачные контейнеры (в контексте продукта) |
